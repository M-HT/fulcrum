# The Fulcrum

[The Fulcrum](https://www.pouet.net/prod.php?which=313 "The Fulcrum") is a demo made by [matrix](https://www.pouet.net/groups.php?which=12233 "matrix").

It was released at [Mekka & Symposium 1998](https://www.pouet.net/party.php?which=26&when=1998 "Mekka & Symposium 1998") where it earned the 1st place in the category 'pc demo'.


This port uses [BASS](https://www.un4seen.com/bass.html "BASS audio library") library or [DUMB](https://dumb.sourceforge.net/ "DUMB - Dynamic Universal Music Bibliotheque") library for music playback.
